1. INTRODUCTION                                        *pi-agent-introduction*

Pi Agent plugin provides integration between Pi Agent AI assistant and Neovim.

Features:
- Toggle Pi Agent terminal with a single keypress
- Floating window and split support
- Auto-reload buffers when Pi Agent modifies files
- Git-aware working directory

Requirements:
- Neovim 0.10+
- Pi Agent CLI (https://github.com/badlogic/pi-agent)

==============================================================================
2. INSTALLATION                                        *pi-agent-installation*

Using lazy.nvim:
>lua
  {
    "aldoborrero/pi-agent.nvim",
    opts = {},
  }
<

==============================================================================
3. COMMANDS                                                *pi-agent-commands*

:PiAgent                                                          *:PiAgent*
    Toggle the Pi Agent terminal window.

:PiAgentContinue                                          *:PiAgentContinue*
    Resume the most recent conversation.

:PiAgentResume                                              *:PiAgentResume*
    Display an interactive conversation picker.

:PiAgentVerbose                                            *:PiAgentVerbose*
    Enable verbose logging mode.

==============================================================================
4. MAPPINGS                                                *pi-agent-mappings*

Default mappings:

  <C-,>        Toggle Pi Agent (normal and terminal modes)
  <leader>pC   Toggle with --continue flag
  <leader>pV   Toggle with --verbose flag

Terminal window navigation:

  <C-h/j/k/l>  Window navigation
  <C-f/b>      Page up/down

==============================================================================
5. CONFIGURATION                                    *pi-agent-configuration*

>lua
  require("pi-agent").setup({
    window = {
      split_ratio = 0.3,
      position = "botright",  -- "botright", "topleft", "vertical", "float"
      enter_insert = true,
      hide_numbers = true,
      hide_signcolumn = true,
      float = {
        width = "80%",
        height = "80%",
        row = "center",
        col = "center",
        relative = "editor",
        border = "rounded",
      },
    },
    refresh = {
      enable = true,
      updatetime = 100,
      timer_interval = 1000,
      show_notifications = true,
    },
    git = {
      use_git_root = true,
    },
    command = "pi",
    command_variants = {
      continue = "--continue",
      resume = "--resume",
      verbose = "--verbose",
    },
    keymaps = {
      toggle = {
        normal = "<C-,>",
        terminal = "<C-,>",
        variants = {
          continue = "<leader>pC",
          verbose = "<leader>pV",
        },
      },
      window_navigation = true,
      scrolling = true,
    },
  })
<
